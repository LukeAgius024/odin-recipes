# odin-recipes
-Odin Recipes
A simple HTML website built as part of The Odin Project Foundations curriculum.
This project demonstrates basic HTML structure, links, lists, images, and file organization.

-Project Overview
The Odin Recipes project is a basic recipe website consisting of multiple pages.
Each page contains a recipe with a description, ingredients list, and preparation steps.

This project focuses on practicing:
*HTML boilerplate
*Headings and paragraphs
*Links and images
*Ordered and unordered lists
*Proper file structure

-Technologies Used
*HTML5
*Git & GitHub

-What I Learned
Through this project, I learned how to:
*Create a multi-page website using only HTML
*Use relative links between pages
*Structure content semantically
*Use Git to track changes and push code to GitHub
